# 遠野洲戦記（とおのすせんき）

> トーノスという名の盆地がある。
> 氷河湖の崩壊によっていまも水に沈む、海と山の交易の要だ。
> 平地人のなかには、トーノスを「呪われた洲（しま）」と呼ぶ者もいる。

システム非依存のTRPGワールドガイド・設定集。遠野・岩手を舞台にしたファンタジー変換。どのTRPGシステムでも使える。

**→ まずは[湖沼人（レイクフォーク）](peoples/lakefolk.md)から読む**
