# 主要NPC

## 大洪水とクーデターの関係者


| 名前     | 所属           | 説明                                                                    |
| ------ | ------------ | --------------------------------------------------------------------- |
| ヒロナガ   | アゾヌム家第14代領主  | 大洪水の時、大きな戦争の遠征に出ていた。水に沈んだ国に帰る場所を失い、ゼンダスに亡命している                        |
| スルガ    | 平地人/クーデター実行者 | ヒロナガの留守中に城を預かっていた三人のひとり。大洪水のどさくさに紛れて政権を奪ったが、民心を得られず統治に失敗した            |
| タンバ    | 平地人/クーデター実行者 | スルガとともに政権を奪った三人のひとり。傀儡政権の一角を担うが、裁定能力を持たない                             |
| サマノスケ  | 平地人/クーデター実行者 | 三人目の政権簒奪者。武力だけが正義となった暗黒時代を体現する人物                                      |
| トシナオ   | ナンバス         | クーデターの黒幕。ナンバスの意向でトーノスを傀儡支配しようとした。背後にある大国の思惑を象徴する                      |
| 清心/ネネ様 | ナンバス出身       | アゾヌム紀439年にトーノスに現れ、暗黒時代を終わらせた女性。人々からは「ねね様」とも呼ばれた。平和のうちにトーノスを治め、清心紀を開いた |
